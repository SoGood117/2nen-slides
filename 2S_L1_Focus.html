Lesson 1: Okinawa

Focus & Try

Focus 1

「～です」「～します」　be動詞・一般動詞の現在形の復習

These islands have very beautiful scenery.

これらの島々にはとても美しい風景があります。

The most famous people in Miyako Island are 20 "Mamoru-kun."

宮古島で最も有名な人々は20人の「まもる君」です。

be動詞：前後の語句が同じ意味になるようにイコールで結んだり、主語の情報や存在を表す。

一般動詞：主語の動作や状態を表す。主語が3人称・単数のときは動詞のうしろに -(e)s をつける。

Focus 2

「～できます」「～するつもりです」など　助動詞の復習

You can enjoy wonderful marine activities.

あなたは、すばらしいマリンアクティビティを楽しむことができます。

助動詞は、動詞の原形の前に置かれ、動詞に意味を付け加える働きをする。

助動詞を主語の前に置くと疑問文、助動詞のうしろに not を置くと否定文になる。

助動詞の例

can（～できます） could（～できました） will（～するつもりです） may（～してもよいです） should（～するべきです） must（～しなければなりません）

Try 1

最も適する語を選び、（　）に入れましょう。

will/can/must

① Ryan ( can ) play the bass guitar very well.

ライアンはベースギターをとても上手に弾くことができます。

② He ( must ) practice very hard every day.

彼は毎日、とても一生懸命練習しなくてはなりません。

③ We ( will ) go to see his concert tomorrow.

私たちは明日、彼のコンサートを見に行くつもりです。

Try 2

最も適する語句を選び、（　）に入れましょう。

important role/consists of/such as

① My band ( consists of ) five members.

私のバンドは5人のメンバーからできています。

② Each member plays an instrument ( such as ) the piano, the acoustic guitar, or the drums.

それぞれのメンバーが、ピアノやアコースティックギターやドラムのような楽器を演奏します。

③ The bass guitar plays an ( important role ) in our band.

私たちのバンドでは、ベースギターが重要な役割をしています。

Try 3

（　）内の語を並べかえましょう。

① I'm ( playing / in / interested ) the bass guitar.

→ I'm interested in playing the bass guitar.

私はベースギターを弾くことに興味を持っています。

② I ( to / be / want ) a famous bassist in the future.

→ I want to be a famous bassist in the future.

私は将来、有名なベーシストになりたいと思っています。

③ So I ( keep / practicing / on ) every day.

→ So I keep on practicing every day.

だから、毎日練習を続けています。
